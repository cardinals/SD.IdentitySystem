#项目说明

    本项目开发的目的有三：

        1、作为一个使用SD.Framework框架开发的项目样板
        
        2、探索基于ASP.NET MVC的前端架构

        3、做一套统一身份认证系统


    注意事项：

        如果编译失败，请安装PostSharp-4.1.15.exe（项目根目录下或附件中有）

        请先修改服务端AppService项目配置文件中连接字符串，数据库会自动生成

        先运行服务端，再运行客户端

        如果服务端运行失败，请检查是不是没装Redis（附件中有）
